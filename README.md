Sistema web integral diseñado para la gestión eficiente de una panadería para uso interno. Este proyecto abarca todo el flujo transaccional del negocio, permitiendo controlar el inventario de productos, registrar ventas, administrar usuarios y visualizar reportes clave en tiempo real.

## 🚀 Tecnologías Utilizadas
* **Frontend:** Angular, TypeScript, HTML5, CSS3.
* **Backend:** Java, Spring Boot.
* **Base de Datos:**  MySQL por MYPHP ADMIN 

## ✨ Características Principales
* **Dashboard de Negocio:** Visualización gráfica del Top 10 de productos más vendidos y métricas clave (usuarios, productos, ventas).
* **Gestión de Inventario:** Control de stock, categorías, precios y estados de disponibilidad.
* **Módulo de Ventas:** Registro detallado del historial de transacciones, métodos de pago y emisión de tickets/comprobantes.
* **Administración de Usuarios:** Gestión de personal con roles específicos (Administrador, Empleado, Jefe de Calidad).
* **Catálogo Visual:** Interfaz amigable para la exploración de productos organizados por categorías (Bebidas, Lácteos, Panes, etc.).
